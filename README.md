# Техническая служба

Статическое веб-приложение для автоматического распределения заявок технической службы.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 4173
```

После запуска приложение будет доступно по адресу `http://localhost:4173`.

## Что внутри

- создание заявки с локацией, заявителем, приоритетом и описанием проблемы;
- несколько ролей и разный интерфейс по роли;
- инженер Александр Скорняков и операционный руководитель видят полный пульт, журнал, команду и аналитику;
- управляющие кофеен и цехов видят только форму заявки и свои заявки со статусами;
- определение категории по ключевым словам: сантехника, электрика, оборудование, холодильники, вентиляция, запчасти, ремонт, качество, координация;
- автоматический выбор исполнителя по специализации и текущей загрузке;
- контроль SLA, просроченных и зависших заявок;
- журнал заявок с изменением статусов;
- аналитика повторяющихся проблем, загрузки команды и прогноза потребности в персонале;
- участники и подразделения взяты из файла `участники.xlsx`, специализации техслужбы заданы как рабочая модель поверх подразделений.
